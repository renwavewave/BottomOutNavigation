# BottomOutNavigation
底部 导航 点击突出

Add it in your root build.gradle at the end of repositories:



	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
	
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.renwavewave:BottomOutNavigation:1.0.0'
	}
